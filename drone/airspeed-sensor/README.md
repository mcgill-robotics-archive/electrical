# Airspeed Sensor

## Inputs
- `+5V`: Voltage (supplied from board)
- `GND`: Ground
- `SCLK`: I2C Clock
- `SDA`: I2C Data

### Components Used
- `U1`: MS4525DO-DS TE Connectivity Airspeed Sensor
- `C1`: 100nF
- `R1 & R2`: 4.7k?
