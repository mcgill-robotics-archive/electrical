# Drive Encoder Board

## Board

Arduino Pro Mini 5V 16MHz (ATMEGA328P)

## Pinout

### Encoder Port 1:

Encoder A+/A- => 1A/1B => Differential Transceiver -> 1Y -> Arduino Pin 2

Encoder B+/B => 2A/2B => Differential Transceiver -> 2Y -> Arduino Pin 6

### Encoder Port 2:

3A/3B => 3Y -> Arduino Pin 3

4A/4B => 4Y -> Arduino Pin 4
