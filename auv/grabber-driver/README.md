# Grabber Driver

__Author:__ Bei Chen Liu

__Description:__ An implementation of brushed DC motor driver with external FET for controlling AUV grabber using DRV8702 from TI.

## Pin Assignments:
### Shared Pins:
* FLT: D2
* SLP: D9 

### Motor 1:
* IN1: D3 
* IN2: D5
* SO: A4
* Encoder A: A0
* Encoder B: A1

### Motor 2
* IN1: D4
* IN2: D6
* SO: A5
* Encoder A: A2
* Encoder B: A3
